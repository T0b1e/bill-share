# Bill Share

A simple bill sharing application built with Svelte and Vite.

**Preview Website:** [https://narongkorn.com/bill-share/](https://narongkorn.com/bill-share/)

## Features
- Add users and split items.
- Support for Service Charge and VAT.
- Automatic debt settlement calculation.
- Modern, responsive UI with Tanuki-inspired aesthetics.
- Dark/Light mode support.

## Tech Stack
- Svelte 5
- Vite
- Tailwind CSS
- Lucide Svelte (Icons)
- Svelte DnD Action (Drag and drop)
